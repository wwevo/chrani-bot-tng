Choose the size and dimensions of your location
Sets the Dimensions of your location
Depending on it's shape, it's either by radius or by defining up to three connecting side-lengths
The radius is set from the position the player is standing at, rectangular shapes from the S/W corner.
This setting can be changed later on, at any time! If a rectangular object is changed into a spherical one, or vice versa, it's geometrical center at ground level will be used as a reference
